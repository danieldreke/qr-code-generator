# QR Code Offline Generator

![Claude](https://img.shields.io/badge/Built_With-Claude-D97757?style=flat&logo=claude&logoColor=D97757) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Open [QR Code Offline Generator](https://danieldreke.github.io/qr-code-generator/)

An offline QR code generator that can be installed on your mobile device. The QR code is generated offline/locally in the browser on your device.

## Features

- Generate QR codes from any text or URL
- Works fully offline after first load
- Installable as a PWA on mobile and desktop
- Save QR code as PNG or SVG
- Copy QR code as PNG or SVG
- Dark / light theme toggle
